# Tesla Mandela Effects

Audio docufiction podcast series. 100-200 standalone episodes, 10,000-13,000 words each, output to ElevenLabs Eleven Turbo v2 TTS for YouTube.

## Episode Writing

Everything you need is in `0. EPISODE FACTORY/`. That folder is self-contained — it has its own CLAUDE.md with read order and instructions. Start there.

## Project Structure

| Folder | Purpose |
|--------|---------|
| `0. EPISODE FACTORY/` | **Active workspace.** Production guide, series bible, voice reference, and all episode drafts. |
| `1. EPISODE SCRIPTS/Old Scripts/` | Archive. Previous episode versions. Do not modify. |
| `2. WRITER AND EDITOR INSTRUCTIONS/` | Archive. Old editing rules superseded by the Production Guide. Do not use for writing. |
| `3. ELEVEN LABS AUDIO/` | Final audio files. |
| `4. EPISODE IMAGES/` | Visual assets for YouTube. |
| `5. SAVED FULL EPISODE FILES/` | Completed production files. |
| `6. EXPORTED WITH KEN BURNS/` | Video exports. |
| `10. TEST AND PLAYGROUND/` | Testing and experimentation. |

## Key Files at Root

- `HANDOFF.md` — Session context for continuity between instances
- `tesla-mandela-decisions.md` — Append-only decision log
- `tesla-mandela-queue.md` — Active task queue
